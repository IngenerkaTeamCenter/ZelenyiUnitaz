Зеленый унитаз

1. Открой main.exe или через code blocks ide открой main.cpp, скомпилируй, запусти.
2. Программа создана для создания идей по планировки дома.
3. (То, что iPad 4 больше духовки) Размер не играет значения в этой программе.   (17-18-16-19-20-16 ipad 18-10-19-16-3-1-13 15-10-12-10-20-1 (A1Z26))
4. Для выбора объекта нажмите на название левой кнопкой мыши
5. Для вставки объекта нажмите на место куда хотите вставить левой кнопкой мыши
6. Для удаления объекта нажмите на место откуда хотите удалить правой кнопкой мыши


Иконки размером 30*30 или кратно (30*60, 120*30...)
